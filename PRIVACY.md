# PanGrab 网盘抓手 · 隐私政策

最后更新：2026-06-29

PanGrab（以下简称"本扩展"）尊重并保护用户隐私。本政策说明本扩展如何处理数据。

## 1. 我们不收集你的个人数据
本扩展**不收集、不上传、不出售**任何个人身份信息。你在本扩展中的收藏数据（网盘链接、分类、标签、备注等）和设置，**仅保存在你本地浏览器中**（chrome.storage.local），不会传输到我们或任何第三方服务器。

## 2. 权限说明
- 主机权限 / activeTab / scripting：在你浏览的网页上识别网盘分享链接（本地处理）。
- storage：在本地保存你的收藏与设置。
- contextMenus：提供右键「收藏此网盘链接」。
- notifications：收藏后在本地弹出提示。

## 3. 网络访问
- **检测失效**：当你主动点击「检测失效」时，本扩展会访问你收藏的链接（即对应网盘资源页的服务器）以判断其是否有效。该请求发往链接本身的服务器，不发往我们，也不在远端存储任何结果。
- **远程配置**：本扩展可能从一个公开地址下载一份 JSON 配置（仅包含推荐/推广的文本与链接），用于展示推荐位；该过程仅为下载数据，不上传你的任何信息。

## 4. 第三方链接
本扩展展示或收藏的网盘分享链接、推广链接由第三方提供，点击后将跳转到第三方网站，其隐私做法由相应网站负责。

## 5. 数据删除
你可随时在管理页删除收藏，或卸载扩展以清除全部本地数据。
